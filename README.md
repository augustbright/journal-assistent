# My Bot PWA

Personal automation tool for daily tasks. Built with React, TypeScript, and Material-UI. Deployed on GitHub Pages.

## Quick Start

```bash
npm install
npm run dev
npm run build
```

## Deploy

```bash
npm run build
git add docs/
git commit -m "Deploy"
git push origin main
```

Available at: `https://augustbright.github.io/journal-assistent/`
